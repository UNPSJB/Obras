Iniciar tramite
====================================
.. image:: _static/profesional/iniciarTramite1.png
.. image:: _static/profesional/iniciarTramite2.png

Esta opción permite dar de alta un nuevo trámite dentro del sistema. Debe completar los siguientes datos del formulario:

- **Tipo de Obra**
- **Medidas** 
- **Domicilio**
- **Parcela** 
- **Circunscripción**
- **Manzana** 
- **Sector** 
- **Propietario (DNI)**

Si el propietario no existe en el sistema debera ingresar los datos en el formulario como se muestra aqui:

   :maxdepth: 3
   
   altaNuevoPropietario
