Resumen de pago 
==============================================
.. image:: _static/propietario/resumenDePago.png
.. image:: _static/propietario/resumenDePago1.png

El resumen de pago se puede imprimir presionando el botón “Imprimir” enmarcado en rosa, y para salir debe apretar el botón “Volver” enmarcado en rojo.
