Mostrar el comprobante de la cuota
==============================================
.. image:: _static/propietario/mostrarElComprobanteCuota.png

Aquí el propietario puede visualizar los comprobantes de pago de cada cuota. 
Para salir de esta página debe apretar el botón “Volver” enmarcado en rojo o apretar el botón azul “Generar Comprobante” para visualizar la factura de esa cuota seleccionada.
