Sectores con mas obras
===============================================

.. image:: _static/director/sectores.png
.. image:: _static/director/sectoresConMasObras.png

Permite ver los sectores con mas obras seleccionando un tipo de obra (obra nueva, ampliacion, etc). Debe apretar boton "Aceptar" enmarcado en rosa para ver el grafico y para volver el boton "Volver" enmarcado en rojo.
